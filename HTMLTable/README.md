- 테이블 태그 종류

__thead -> 테이블 헤더(제일 윗줄을 의미) th라고도 씀__     
__tbody -> 테이블 헤더를 제외한 줄__    
__tr -> 테이블의 행__      
__td -> 테이블의 데이터__     
__rowsapn ="3" -> 행의 3부분을 합친다, 줄이 없어진다__      
__colspan -> 열의 부분을 합친다__   
__footer ->테이블에 대한 설명(테이블 밖에다 사용해야함)__
